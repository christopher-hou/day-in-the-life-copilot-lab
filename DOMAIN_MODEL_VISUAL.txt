DEPARTMENT:
  ★ Name:        Required, 3-50 chars, start with letter
                 Cannot contain: Test, Demo, Sample, Temp
  ★ Budget:      0-10M, varies by type:
                   Academic: 50K-5M
                   Admin: 100K-10M
                   Research: 200K-20M
  ★ StartDate:   Required, not more than 5 years in future
  
COURSE:
  ★ Credits:     0-5 range
  ★ Title:       Required, 3-50 chars
  
STUDENT:
  ★ EnrollmentDate: Required, range 1753-9999
  
INSTRUCTOR:
  ★ HireDate:    Required, range 1753-9999


================================================================================
                         DATA INTEGRITY RULES
================================================================================

CASCADE DELETES:
  • Delete DEPARTMENT → Delete all its COURSES
  • Delete COURSE → Delete all its ENROLLMENTS & COURSEASSIGNMENTS
  • Delete STUDENT → Delete all its ENROLLMENTS
  • Delete INSTRUCTOR → Delete COURSEASSIGNMENTS & OFFICEASSIGNMENT

REQUIRED FOREIGN KEYS (NOT NULL):
  • Course.DepartmentID
  • Enrollment.StudentID
  • Enrollment.CourseID

OPTIONAL FOREIGN KEYS (NULL allowed):
  • Department.InstructorID (may not have administrator)
  • Instructor.OfficeAssignmentID (implied by shared PK)

UNIQUE CONSTRAINTS:
  • Course.CourseID (unique identifier)
  • Enrollment: (StudentID, CourseID) composite unique
  • CourseAssignment: (InstructorID, CourseID) composite unique
  • OfficeAssignment.InstructorID (one-to-one via shared PK)

CONCURRENCY CONTROL:
  • Department.RowVersion [Timestamp]
    → Prevents lost updates
    → Throws ConcurrencyException if version mismatch


================================================================================
                       NAVIGATION EXAMPLES
================================================================================

Get student's courses and grades:
  student.Enrollments          ← Collection<Enrollment>
  ├─ [0].Course                ← Navigate to Course
  ├─ [0].Course.Title          ← Get course name
  ├─ [0].Grade                 ← Get grade (A|B|C|D|F|null)
  └─ [n]...                    ← Multiple enrollments

Get course's instructors:
  course.CourseAssignments     ← Collection<CourseAssignment>
  ├─ [0].Instructor            ← Navigate to Instructor
  ├─ [0].Instructor.FullName   ← Get instructor name
  └─ [n]...                    ← Multiple instructors

Get department's courses and students:
  dept.Courses                 ← Collection<Course>
  ├─ [0].Enrollments           ← Get enrollments in course
  │   ├─ [0].Student           ← Navigate to Student
  │   ├─ [0].Student.FullName  ← Get student name
  │   └─ [m]...                ← Multiple enrollments
  └─ [k]...                    ← Multiple courses

Get instructor's office:
  instructor.OfficeAssignment  ← OfficeAssignment (nullable)
  └─ .Location                 ← Get office location
